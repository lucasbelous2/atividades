Existem vários tipos de campos de entrada disponíveis, incluindo:

  • texto: para entrada de texto em uma linha
  • senha: para entrada de texto mascarado (oculta os caracteres de entrada)
  • e-mail: para entrada de endereço de e-mail
  • url: para entrada de URL
  • número: para entrada numérica
  • data: para entrada de data
  • hora: para entrada de hora
  • caixa de seleção: para entrada de escolha múltipla
  • radio: para entrada de escolha única

O método post é usado para enviar dados do formulário para o servidor/api. Quando o usuário clica no botão enviar, o navegador envia os dados do formulário como uma solicitação HTTP para o servidor usando o método post. O servidor pode então processar os dados e retornar uma resposta ao cliente.
